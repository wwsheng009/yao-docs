## 设置控件的必输，使用编辑控件的配置属性`itemProps`

```json
{
  "名称": {
    "bind": "name",
    "edit": {
      "props": {
        "itemProps": {
          "rules": [{ "required": true }]
        },
        "placeholder": "请输入 名称"
      },
      "type": "Input"
    },
    "view": {
      "type": "Text"
    }
  }
}
```
