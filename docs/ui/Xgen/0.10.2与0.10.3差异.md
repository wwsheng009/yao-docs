# yao 应用引擎模板

注意：0.10.2 与 0.10.3 引擎并不兼容，所以使用的模板也不一样。

## 菜单

菜单的数据格式不一样。可以对比以下两个文件
0.10.2 的菜单输出是一个数组
0.10.3 的菜单输出是一个对象

## form/table dsl 不一样

2 个版本中的 form/table dsl 中的 action 对象结构不一样

## 模板使用

0.10.2 的请直接使用官方的模板

```sh
mkdir your-app
cd your-app
yao start
```

0.10.3 的请使用以下模板

```sh
git clone https://github.com/wwsheng009/yao-init-0.10.3
cd yao-init-0.10.3
yao start
```
