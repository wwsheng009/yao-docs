# 聊天布局配置

在1.0版本中，聊天布局配置是通过Xgen的DSL语法来实现的。

有两种布局，分别是`Chat`和`Admin`,默认是`Admin`布局,可以通过`layout`字段来切换。

```json
{
  "adminRoot": "admin",
  "optional": {
    "hideNotification": true,
    "hideSetting": false,
    "layout": "Chat"
  }
}
```

adminRoot字段是管理端入口地址的根路径，如`http://localhost:5099/admin`。

聊天入口地址是adminRoot地址加上/chat,比如这里配置的adminRoot是admin,那么访问地址是`/admin/chat`，如`http://localhost:5099/admin/chat`
